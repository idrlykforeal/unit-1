## Last digit

Given an integer number, print its last digit.

```.py
# Read an integer:
a = int(input())
print(a%10)

``` 


## Tens

Given an integer. Print its tens digit.

```.py
# Read an integer:
number = int(input())

number = number//10 # integer division
answer = number%10 
print(answer)

```
